【护肤小知识】
日常护肤顺序：
- 早上：洁面 → 爽肤水 → 精华 → 乳液/面霜 → 防晒
- 晚上：卸妆 → 洁面 → 爽肤水 → 精华 → 眼霜 → 面霜

不同肤质注意事项：
- 干皮：注重保湿，用滋润型产品，少用皂基洁面
- 油皮：适度清洁，控油但不能过度，T区重点护理
- 混合皮：分区护理，T区控油，脸颊保湿
- 敏感肌：精简护肤，选无酒精无香精的产品

防晒很重要：
- 紫外线是皮肤老化的元凶，无论阴晴都要涂防晒
- 防晒要涂够量，大概一元硬币大小
- 户外活动要两小时补涂一次
- SPF30日常够用，户外用SPF50

【穿搭小技巧】
色彩搭配原则：
- 同色系搭配：高级感，不易出错
- 邻近色搭配：和谐舒服
- 互补色搭配：活泼吸睛，但要控制比例
- 黑白灰是百搭色，不知道怎么穿就用它们

不同身材穿衣建议：
- 梨形身材（上身瘦下身胖）：突出上半身，A字裙遮胯
- 苹果身材（肚子大）：V领拉长线条，高腰线显腿长
- 沙漏身材（肩胯宽腰细）：收腰款最合适
- H型身材（直筒）：制造腰线，增加层次感

【口红挑选指南】
根据肤色选口红：
- 冷白皮：蓝调红、玫红、浆果色、豆沙色
- 暖黄皮：橘调红、砖红、番茄红、枫叶红
- 小麦色：正红、姨妈红、裸土色、吃土色

经典色号类型：
- 正红色：气场全开，正式场合
- 豆沙色：日常通勤，温柔知性
- 烂番茄色：显白百搭，黄皮友好
- 奶茶色：素颜可用，伪素颜感

【恋爱小贴士】
情感沟通技巧：
- 用"我"开头表达感受，而不是指责对方
- 倾听比说更重要，先理解再被理解
- 吵架时不说伤人的话，气头上先暂停
- 定期制造小惊喜，仪式感很重要

如何保持新鲜感：
- 一起尝试新事物，比如学跳舞、做饭
- 给对方空间，不要过度依赖
- 保持自己的社交圈和兴趣爱好
- 多夸奖对方，细节的赞美最动人
- 偶尔写写小纸条，制造小浪漫

【健康生活小建议】
睡眠建议：
- 成年人需要7-8小时睡眠
- 睡前1小时不看手机，蓝光影响褪黑素分泌
- 固定作息时间，周末也别差太多
- 午睡20分钟最佳，超过1小时反而累

喝水小知识：
- 每天喝1.5-2升水
- 不要等渴了再喝，那时已经缺水了
- 早上空腹喝一杯温水，唤醒肠胃
- 奶茶饮料不能代替水

【居家小妙招】
收纳技巧：
- 垂直收纳：利用墙面空间
- 分类收纳：同类物品放一起
- 透明收纳盒：一眼看到内容
- 断舍离：一年没用过的东西可以考虑扔掉

清洁小窍门：
- 白醋+小苏打：厨房油污克星
- 柠檬酸：去除水垢神器
- 酒精：擦玻璃不留痕
- 牙膏：擦银饰、擦鞋边都好用
