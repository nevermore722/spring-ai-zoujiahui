【Java 编程知识】
Java 17 是长期支持版本，引入了很多新特性：
- 文本块：用三个双引号可以写多行字符串，不用再拼\n了
- Record 类：可以用一行代码定义数据类，自动生成构造方法和 getter
- Switch 表达式：switch 可以直接返回值，不用写 break
- instanceof 模式匹配：判断类型的同时可以直接赋值

代码重构原则：
- DRY：Don't Repeat Yourself，避免重复代码
- KISS：Keep It Simple, Stupid，保持简单
- SOLID：面向对象设计的五个基本原则
- 重构要小步快跑，每次只改一点点，保证测试通过

调试技巧：
- 断点调试比打印日志更高效
- 条件断点可以只在特定条件下暂停
- 异常断点可以在抛出异常时自动暂停
- 日志要分级别：DEBUG、INFO、WARN、ERROR

【Spring Boot 知识】
Spring Boot 3.x 要求 Java 17 以上
自动配置是 Spring Boot 的核心，根据 classpath 的依赖自动配置 Bean
Starter 依赖简化了 Maven 配置，一个 starter 包含所有需要的依赖
Spring AI 是 Spring 官方推出的 AI 应用开发框架
Spring AI 支持多种大模型：OpenAI、Ollama、DeepSeek 等

【Git 常用命令】
git status：查看当前状态，最常用的命令
git add .：添加所有修改到暂存区
git commit -m "message"：提交修改
git push：推送到远程仓库
git pull：拉取远程更新
git branch：查看分支
git checkout -b feature：创建并切换到新分支
git merge：合并分支，记得先切到目标分支

遇到合并冲突怎么办：
1. git status 查看冲突文件
2. 手动编辑冲突文件，删除 <<<<<<< ======= >>>>>>> 标记
3. git add 标记为已解决
4. git commit 完成合并

【游戏开发小知识】
游戏设计原则：
- 心流体验：难度要适中，太简单无聊，太难劝退
- 反馈机制：玩家操作要立刻有反馈
- 奖励曲线：初期奖励频繁，后期逐渐稀疏
- 新手引导：要在游戏中自然教学，不要弹大段文字

游戏性能优化：
- 对象池：频繁创建销毁的对象用池化
- LOD：远处物体用低模
- 遮挡剔除：不渲染看不见的物体
- 纹理压缩：减少显存占用

【程序员日常小贴士】
- 颈椎不好的话，把显示器垫高到视线水平
- 眼睛疲劳时，试试 20-20-20 法则：每20分钟看20英尺外20秒
- 手腕疼可能是鼠标手，换个垂直鼠标或者用腕托
- 久坐是最大的健康杀手，设个闹钟每小时站起来活动一下
- 黑咖啡比红牛健康，提神效果也不错
- 写代码前先想清楚，写注释不是浪费时间
